# Быстрый старт MPInformer

## Минимальные шаги для запуска

1. **Создайте виртуальное окружение и установите зависимости:**
   ```powershell
   python -m venv venv
   .\venv\Scripts\Activate.ps1
   pip install -r requirements.txt
   ```

2. **Настройте переменные окружения:**
   ```powershell
   copy .env.example .env
   # Откройте .env и заполните токены API
   ```

3. **Инициализируйте базу данных:**
   ```powershell
   python -m app.db.init_db
   ```

4. **Запустите приложение:**
   ```powershell
   python main.py
   ```

## Что происходит после запуска?

- Приложение запускает веб-сервер на http://localhost:8000
- Планировщик задач автоматически начинает проверять заказы каждые 60 минут (или как указано в `SCHEDULER_INTERVAL_MINUTES`)
- Отчеты автоматически отправляются в указанный Telegram чат

## Получение токенов API

### Ozon:
1. Личный кабинет продавца → Настройки → API
2. Скопируйте `Client-Id` и `API-Key`

### Wildberries:
1. Личный кабинет продавца → Настройки → Доступ к API
2. Скопируйте токен

### Telegram Bot:
1. Найдите [@BotFather](https://t.me/botfather)
2. `/newbot` → следуйте инструкциям
3. Скопируйте токен
4. Для получения Chat ID: добавьте [@userinfobot](https://t.me/userinfobot) в чат → `/start` → скопируйте `Id`

## Проверка работы

1. Откройте http://localhost:8000 в браузере
2. Проверьте логи в консоли - должны быть сообщения о запуске
3. Дождитесь первого автоматического отчета или отправьте команду `/report` боту

## Изменение интервала проверки

Измените значение `SCHEDULER_INTERVAL_MINUTES` в файле `.env` (в минутах).
